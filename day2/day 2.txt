5 types of tokens

keywords- reserved words in python (if, elseif, elif)
identifiers- Names given to variables, functions, classes
Literals- constant values assigend to variables(10, "HeLLo")
operators- symbols that perform operations(+,-,*,/)
punctuators-- symbols like (), {}, []

python is case sensitive and it can include letters from a to z, A to Z and 0-9, underscore.
It cannot start with a digit and cannot use python keywords as names. Avoid using special characters @,#,$

Swapping is allowed in python without using a temporary variables
a,b=5,10
a,b=b,a
print(a.b)

deleting variables
use the del keyword to delete a variable
